# Learning to Code a Chess Game from Scratch
### 🎯 Core Concept
The core concept of this project is to determine if an average person with no coding experience can successfully build a working chess game. The author, with no background in computer science or STEM, attempts to create a chess game using Python, starting with a 1-hour Python quick start guide on LinkedIn Learning.

### 📖 Detailed Breakdown
The author begins by rendering a chessboard using the Turtle package, a Python library used to teach kids the basics of programming. They encounter several challenges, including:
* Creating a chessboard with alternating colors
* Making pieces move around on the board
* Preventing pieces from occupying the same square
* Implementing movement restrictions for each piece (e.g., pawn, knight, bishop, rook, queen, king)
* Handling special rules, such as castling and en passant
* Preventing the king from moving to illegal squares when in check

The author uses various techniques, including:
* Hard-coded dictionaries to store coordinates and piece positions
* Conditional logic to implement movement rules
* Loops within loops to check for legal moves and calculate enemy team's moves

### ⚙️ Workflow / Steps
The author's workflow involves:
1. Setting up the project with a 1-hour Python quick start guide
2. Rendering a chessboard using the Turtle package
3. Creating pieces and implementing movement rules
4. Handling special rules, such as castling and en passant
5. Preventing the king from moving to illegal squares when in check
6. Testing and debugging the game

### 🚀 Applications
The applications of this project include:
* Demonstrating the feasibility of learning to code a complex game like chess from scratch
* Showcasing the importance of persistence and problem-solving in coding
* Providing a unique perspective on the challenges and rewards of learning to code

### 📝 Summary
In summary, the author successfully creates a working chess game from scratch, despite having no prior coding experience. The project highlights the challenges and rewards of learning to code and demonstrates the importance of persistence and problem-solving in overcoming obstacles. The author's approach, although not always elegant, ultimately leads to a functional chess game that can be played locally.
